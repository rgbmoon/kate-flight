## Install

1) npm install
2) npm install netlify-cli
3) netlify login
4) netlify init
5?) maybe you need install some deps for lambda(like axios)

## Start 

1) netlify dev

## Deploy

Netlify ci/cd binded to main btanch. Deploy will start automatically.
